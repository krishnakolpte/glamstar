## üì∏ Key Features

### 1. Retro-Cinematic Hero Sections
Uses **Framer Motion** `useScroll` and `useTransform` to create a slow-zoom effect on archival imagery, mimicking a high-end fashion documentary.

### 2. The Visionary (About Page)
An editorial layout featuring the founder's note with overlapping grids, signature typography, and high-contrast grayscale-to-color transitions.

### 3. Season Retrospectives (Season 2)
- **Champion Showcase:** Interactive cards for winners with glassmorphism badges.
- **Celebrity Spotlight:** A responsive, large-scale typography reveal for guest icons.
- **Venue Architecture:** A data-driven summary of the event scale and locations.

### 4. Fluid Responsiveness
Implemented using CSS `clamp()` functions to ensure typography looks as stunning on an iPhone as it does on a 4K studio monitor.
